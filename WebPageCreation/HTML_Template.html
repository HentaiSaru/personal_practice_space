HTML元素參考

Metadata 元數據元素

title 標頭

用於定義該網頁於分頁欄顯示的以及提供給瀏覽器辨識的名稱。

link 連結

用於網頁與資源之間的連結並定義與資源之間的連結關係(rel屬性定義關係)。

meta 數據

用於定義該網頁的數據資訊並提供給瀏覽器或其它web服務使用。

script 腳本

用於加載JS腳本，該元素內部默認可以輸入JS語法運行邏輯但不建議。

base 根

用於定義該網頁所有的相對URL所參考的位置。

Layout 佈局元素

header 標題

用於頁面的開頭起始部分。

nav 導航

用於頁面的導航分類條目部分。

main 主體

用於頁面的核心主體內容。

section 部分

用於頁面的部分內容(沒有其它適合使用的元素時才使用這個)。

aside 旁白

用於非重點內容與主內容較無關之部分。通常做為側邊攔使用。

footer 頁腳

用於頁面的結尾頁腳部分的內容。

article 文章

用於如文章或社群貼文等重複出現且獨立的內容。

details 細節欄

用於建立細節欄(可以打開或收起的內容欄位)。

summary 概括

用於建立細節欄的名稱，該元素必須作為details的子元素使用。

細節欄 細節欄的文本內容

figure 突出內容

用於建立一段內容中的突出部分，例如: 文章中的圖片、表格、代碼段...等等。該元素呈現縮排。

figcaption 突出內容說明

用於對figure附帶的網頁元素的附加說明。該元素必須作為figure的子元素使用，且必須作為第一或是最後一個子元素。

menu 菜單

實驗中的功能元素。文章連結

Paragraphs 段落元素

div 空區塊元素

不具任何特殊性質的通用區塊元素。適用於將多個相關元素包裹，使用id或class屬性來分類。

span 空內聯元素

不具任何特殊性質的通用內聯元素。適用於內容的編排，常使用id或class屬性來分類。

h1 一級標題

h2 二級標題

h3 三級標題

h4 四級標題

h5 五級標題
h6 六級標題

p 文字區塊

用於顯示純文字的區塊元素。

a 超連結

用於建立通往其他URL的超連結。

測試連結 測試前往頁面最下方 測試信箱連結

br 跨行

用於在文本中跨行，像
這樣。

hr 分隔線

跨行並顯示一個線條。

Formatting 格式化元素

b 粗體

使內容呈現粗體。

i 斜體

使內容呈現斜體。

small 縮小

使內容呈現縮小。

q 引言

使內容前後加上引號。

s 刪除線

使內容加上刪除線。

u 下劃線

使內容加上下劃線。

bdo 文本順序

用於改變文本的文字排序，以dir屬性來更改方向。值ltr = 左至右。值rtl = 右置左。

bdi 文本隔離

用於將特定內容獨立出來與其它文本區隔開，此元素通常用於那些可能破壞排版的特殊語言格式。

strong 強烈

用於標註重點內容。內容呈現強烈

em 重點

用於標註重點內容。內容呈現斜體

mark 標記

用於標註重點內容。內容的背景呈現高亮

abbr 縮寫

用於註記重點內容。使用title屬性使得焦點在該元素上會出現一個區塊顯示註記文字。

blockquote 區塊引用

用於標註一段內容的引用出處或是詳細解釋。內容呈現縮排。使用cite屬性附上關於這段內容的相關連結。

cite 標籤

用於呈現具有特殊涵義的名詞，例如:人名、書名、歌名、作品名...等等的。內容呈現斜體。

address 地址

用於存放如:連絡電話、住址、社交媒體帳號...等等的個人資訊內容。內容呈現斜體。

我的Gmail:s9613010@gmail.com

time 時間

用於標註時間，該元素可被編譯器辨識時間格式。2022-09-01T12:00

sup 上半部文字

用於呈現像數學的次方或是腳註的上半部文字。22 * 22 =16

sub 下半部文字

用於呈現像化學式的下半部文字。H2SO4

ruby 拼字註記

用於呈現東亞文字的拼音註記。 日 ni 本 hon 語 go

其餘不多加說明的元素

ins=添加內容、del=移除內容、kbd=輸入按鍵、var=數學公式、code=程式碼、samp=計算機程序樣本、data、dfn、wbr。

List 列表元素

dl 基本列表

用於建立一個基本列表結構。使用dt元素呈現列表標題，使用dd元素呈現列表內容。

基本列表標題
基本列表內容1
基本列表內容2

ul 無序列表

用於建立一個無序列表結構。使用li子元素呈現列表內容，li元素前頭帶有預設的黑點標記。

  • li 無序列表內容1
  • li 無序列表內容2

ol 有序列表

用於建立一個有序列表結構。使用li子元素呈現列表內容，li元素前頭帶有預設的數字記號(可用start屬性改變)。使用reversed屬性可以顛倒記號順序。

  1. li 有序列表內容1
  2. li 有序列表內容2

Table 表格元素

table 表格

用於建立一個表格結構。

caption 表格標題

用於建立表格的標題。該元素必須作為table的子元素使用。

colgroup 表格列組

用於建立表格的列組。該元素必須作為table的子元素使用。使用col子元素代表表格的每一列。如要對表格的第三列做編輯，則需要三個col子元素並對第三個做屬性編輯。

col 表格列

用於建立表格的一列。該元素必須作為colgroup的子元素使用。

tr 表格行

用於建立表格的一行

th 表格行標題

用於建立表格行的標題。該元素必須作為tr的子元素使用，通常只使用於表格的第一行。

td 表格行內容

用於建立表格行的內容。該元素必須作為tr的子元素使用。

其餘未說明的元素

thead、tbody、tfoot。這些元素只是為了表格分類方便，不一定要用。

東方角色表格。
姓名 種族 年齡
靈夢 人類 少女的年齡

Embedded Content 嵌入內容元素

iframe 框架

用於再嵌入一個網頁區域(顯示內容根據URL決定)。設定寬度高度的屬性來限制嵌入網頁的顯示範圍。

object 物件

用於嵌入資源內容。必須指派type屬性，type的值為 MIME type。

source 資源

用於為一個媒體元素提供多個資源並以屬性設定條件讓使用者端根據條件判斷最適用的資源並顯示，該元素必須作為picture、audio、video的子元素使用。

Image 圖像元素

img 圖片

用於嵌入圖片內容。

picture 響應式圖片

用於建立響應式圖片(瀏覽器根據source的條件選擇適合的圖片顯示)，必須含有一個img子元素(必須是最後的子元素)，Source子元素可多個也可無。

map 圖像映射

用於建立一個圖像映射(滑鼠點擊可以回饋的區域)，必須含有area子元素和配對一個img元素，map使用name屬性(img元素使用usemap屬性，兩屬性的值必須相同)。

area 映射區域

和map元素一起使用，用於定義映射的區域範圍，該元素必須作為map的子元素使用。

Multimedia 多媒體元素

audio 音訊

用於嵌入音訊媒體內容。

video 影片

用於嵌入影片媒體內容。

track 軌道

用於嵌入媒體內容的字幕。該元素必須作為audio、video的子元素使用

Forms 表單元素

form 表單

用於建立一個表單結構。使用action屬性指派提交表單的位置(php)。使用target屬性指派以何種方式開啟提交表單後顯示的視窗

使用method屬性指派提交表單要執行的HTTP請求操作。使用autocomplete屬性可以決定下次在開啟表單是否要自動填入上次用戶所提交的值

fieldset 表單集區

用於建立表單集區，將表單分區分類。

legend 集區標題

用於建立表單集區的標題。該元素必須作為fieldset的子元素使用。

label 標籤

用於建立表單元素的簡介說明。使用for屬性填入與之對應的表單元素的id來關聯兩者會較OK。

input 輸入欄

用於建立輸入欄。必須指派type屬性，type屬性會影響可輸入的內容類別。value屬性代表這個輸入框會提交給表單的值。 type屬性參考

output 輸出欄

用於建立輸出欄。參考文章

datalist 資料列表

用於建立資料列表(不可見)。必須含有option子元素。必須指派id屬性。input元素使用list屬性並輸入資料列表的id即可取得該資料列表並根據選項填入資料。

select 選擇列表

用於建立選擇列表(可見)。必須含有option子元素。

optgroup 選項分類

用於建立選項列表中的選項分類標頭。該元素必須作為select的子元素使用。必須指派label屬性。該元素不會被視為可選的選項。

option 選項

用於建立列表中的選項。該元素必須作為datalist、select的子元素使用。必須指派value屬性，value屬性代表提交給表單的值。若使用selected屬性則該選項設定為預設選項。

textarea 文本區

用於建立文本區。使用cols、rows屬性可分別指定寬度和行數。使用maxlength屬性可限制最大的文本字數。

meter 儀表

用於建立儀表(狀態條)。必須指派min、max、value屬性。分別代表最小值最大值當前值

progress 進度條

用於建立進度條。必須指派max、value屬性。

button 按鈕

用於建立按鈕。必須指派type屬性(button、reset、submit)。

dialog 對話框

用於建立對話框。該元素建議建立一個含有button的form子元素，form使用method屬性並設定dialog，可使button變成用於關閉對話框視窗的功能按鈕。

Web Components 網頁元件元素

template 模板

用於建立模板。template並不會被網頁渲染出來，而是做為參考模板供JS腳本取用，被取用後才會渲染出內容。

slot 插槽

用於建立插槽。參考文章

其它元素

noscript 無腳本

用於在腳本被關閉或無法使用時執行。

canvas 畫布

用於建立畫布，可在網頁上供使用者繪製圖形，但該元素只是提供容器並沒有實質功能，功能需由JS腳本撰寫。

svg 向量圖

用於繪製向量圖(使用該元素並不是要顯示SVG圖檔，而是直接繪製向量圖)。

math 數學算式

過於複雜且幾乎不會用到，不詳細說明...

acronym、applet、bgsound、big、blink、center、content、dir、embed、font、frame、frameset、image、keygen、marquee、

menuitem、nobr、noembed、 noframes、param、plaintext、rb、rtc、shadow、spacer、strike、tt、xmp。
